# 📝 TaskFlow

Aplicación de gestión de tareas desarrollada con React + TypeScript, enfocada en una experiencia de usuario minimalista y arquitectura escalable.

## 🚀 Demo

🔗 https://taskflow.vercel.app

## ✨ Features

* Crear, editar y eliminar tareas
* Marcar tareas como completadas
* Filtros (todas / completadas / pendientes)
* Edición inline con Enter y blur
* Confirmación al eliminar
* Persistencia con localStorage

## 🧠 Tecnologías utilizadas

* React
* TypeScript
* Context API
* Tailwind CSS
* Vite

## 🎯 Enfoque del proyecto

Este proyecto fue desarrollado con énfasis en:

* Manejo de estado global con Context API
* Eliminación de props drilling
* Separación de lógica y UI
* UX minimalista y funcional
